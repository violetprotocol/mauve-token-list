# Mauve Token List

Token list for the Mauve DEX.

Tokens supported:

- Wrapped Ether
- Wrapped BTC
- USDC
- USDT
- DAI

See [here](https://github.com/Uniswap/token-lists) for more details.

## Usage

To validate the list run:

```bash
$ yarn validate
```

If the list does not conform to the defined schema, it will fail.
